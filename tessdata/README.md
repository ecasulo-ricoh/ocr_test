# Tesseract Language Data

Para que funcione el OCR, necesitas descargar los archivos de entrenamiento de idiomas:

## Descargar archivos requeridos:

1. **Inglés**: https://github.com/tesseract-ocr/tessdata/raw/main/eng.traineddata
2. **Español**: https://github.com/tesseract-ocr/tessdata/raw/main/spa.traineddata

## Instrucciones:
1. Descargar ambos archivos
2. Colocarlos en esta carpeta `tessdata/`
3. Los archivos deben quedar como:
   - `tessdata/eng.traineddata`
   - `tessdata/spa.traineddata`
